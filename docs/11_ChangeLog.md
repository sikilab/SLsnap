# Change Log

## 2026-05-07

- 移除工具栏撤销/重做按钮，撤销和重做仅保留 `Ctrl+Z` / `Ctrl+Y` 快捷键。
- 调整马赛克颜色逻辑：本身颜色控制深色块，描边颜色不再影响马赛克。
- 工具栏新增小手工具，放大截图后可拖动贴片内截图内容和标注视图位置。
- 小手平移增加边界限制，避免截图完全拖出可视区；双击小手工具可将截图重新水平垂直居中。
- 优化贴片外框拖拽调整尺寸时的重绘方式，降低 resize 闪屏。
- 修复色盘宽度不足导致白色或“无”选项被裁切的问题。
- 鼠标滚轮缩放改为缩放贴片内的截图内容，不再同步改变贴片外框尺寸。
- 贴片外框支持通过窗口边缘拖动调整尺寸，截图未覆盖区域使用灰底填充。
- 修复窗口初始化尺寸变化早于画布创建时触发的空引用异常。
- 统一文字填充和描边的绘制路径与字号换算，避免红色填充文字和描边文字尺寸不一致。
- 默认描边颜色改为“无”，避免新建文字默认出现黑色描边，被误认为双层文字。
- 编辑文字时将当前文字对象临时移出标注列表，提交后再插回原位置，避免编辑态双层绘制。
- 文字编辑时禁止画布 `MouseEnter` 抢焦点，避免鼠标移出文本框后无法继续输入。
- 字号控件范围固定为 12-30，文字绘制使用像素字号模型，贴近网页 CSS 字号视觉。
- 调整字号不再主动改变文本框高度，仅在文字内容实际超高时向下增长。
- 文字工具激活时，若已有文字正在编辑或已有选中对象，点击空白只提交/清选；只有无编辑无选择时才新建文本框。
- 工具栏图标按钮从原生 Button 改为自绘 Control，避免初始绘制残留边框、小格子和外发光。
- 色盘按钮改为自绘圆角色块，并扩大面板宽度保证颜色完整展示。
- 文字编辑框字体改为像素单位，和最终绘制字号保持同一缩放模型；移除鼠标移出导致的自动提交，仅在点击贴片其他位置时提交。
- 工具栏按钮未选中时去除独立边框和小格子感，仅选中工具显示蓝色背景。
- 字号选择控件改为自绘紧凑步进控件。
- 贴片右上角复制、保存、关闭按钮改为粗黑体风格字体。
- 修复手绘标注缩放时点集被重复变形导致的累计放大和漂移。
- 文字编辑框改为无边框，并以文字位置下的截图像素作为背景色，避免白底块；字号在创建编辑框前按画布缩放计算。
- 色盘改为横向色块面板样式；撤回/重做图标改为弧形回转箭头。
- 选中标注后支持方向键按 1 像素移动，`Shift + 方向键` 按 10 像素移动，并纳入撤回/重做历史。
- 固定色盘为 red / orange / yellow / green / blue / purple / gray / black / white，描边色盘额外支持“无”。
- 圆形绘制后支持通过四边控制点单向拉伸为椭圆。
- 统一工具栏图标按钮的背景和间距，未选中时不再显示独立色块底，撤回/重做图标重新绘制。
- 工具栏移到底部，撤回/重做并入同一排图标按钮；窄贴片下按最小宽度和折行策略避免工具栏溢出隐藏。
- 工具栏图标增加中英文悬停提示。
- 标注控制点扩展为四角和四边，角点拖动时支持按住 `Shift` 等比缩放。
- 手绘标注和文字框缩放改用稳定内容边界，减少调整大小时的漂移。
- 项目输出架构调整为 x64，消除缺少 reference assemblies 以外的 MSIL/AMD64 架构不匹配警告。
- 修复标注图形拖动控制点调整大小时的漂移问题，并允许拖拽中坐标贴边继续更新。
- 选中标注后支持按 `Delete` 删除。
- 文字层支持选中后通过工具栏字号控件调整文字大小。
- 文字输入改为在贴片画布上直接输入，支持文字框宽度控制自动折行、顶部对齐和内容超高时向下增长。
- 根据贴片标注交互反馈调整：马赛克改为 100% 不透明遮盖，工具栏绘制工具改为图形图标。
- 移除单独 Move 按钮，改为点击已有标注直接选中、拖动移动，并通过控制点调整大小。
- 箭头选中后支持拖动两端控制点调整长度和方向。
- 支持 `Shift` 点击多选多个图形或文字；两个色盘分别调整选中标注的本身颜色和描边颜色。
- 贴片窗口新增标注工具：空心方形、空心圆形、手绘圈、箭头、文字和马赛克遮盖。
- 文字标注支持设置文字大小、文字颜色和描边颜色。
- 已绘制标注支持移动调整位置，并支持 `Ctrl+Z` 撤回、`Ctrl+Y` 重做。
- 复制、保存和保存全部贴片改为输出包含标注的合成图片。
