# SLSnap 产品使用介绍 / Product Usage Guide

## 中文版

### 1. 产品简介

SLSnap 是一个 Windows 桌面截图贴图工具。

它可以帮助你快速截取屏幕上的局部区域，并把截图直接变成一个始终置顶的桌面贴图窗口，方便你在处理文档、核对数据、比对设计稿、录入信息时随时参考，不需要反复切换窗口。

### 2. 适用场景

- 对照截图内容填写表单或录入系统
- 同时比对多份图片、表格、网页或设计稿
- 截图后临时悬浮在桌面上做参考
- 快速复制截图到聊天工具、文档或邮件
- 将截图保存到本地，作为记录或反馈材料

### 3. 启动方式

启动 SLSnap 后，程序会在系统托盘中常驻运行。

- 双击托盘图标：立即开始截图
- 右键托盘图标：打开功能菜单
- 默认全局截图快捷键：`Ctrl+Alt+A`

> 说明：如果默认快捷键与你电脑中的其他软件冲突，可以在设置中修改。

### 4. 快速上手

#### 4.1 开始截图

你可以通过以下任一方式开始截图：

- 按下全局快捷键 `Ctrl+Alt+A`
- 双击系统托盘中的 SLSnap 图标
- 右键托盘图标后选择“截图”

开始截图后，屏幕会进入选区模式。

#### 4.2 选择截图区域

- 按住鼠标左键拖出需要截图的区域
- 拖动选区边缘或角点，可以调整截图范围
- 拖动选区内部，可以整体移动选区
- 按住 `Shift` 调整大小时，可保持比例
- 按 `Enter` 可确认截图
- 按 `Esc` 可取消截图
- 双击已选区域，也可直接确认截图

#### 4.3 生成桌面贴图

确认截图后，SLSnap 会立即生成一个桌面贴图窗口。贴图会保持置顶，便于你边看边操作。

### 5. 贴图窗口怎么用

每个贴图窗口都可以单独操作。

#### 5.1 移动贴图

按住贴图顶部区域拖动，即可移动位置。

#### 5.2 缩放贴图

将鼠标移到贴图图片区域，滚动鼠标滚轮即可放大或缩小。

#### 5.3 复制贴图

点击贴图窗口上的“复制”按钮，可将当前截图复制到剪贴板，方便直接粘贴到聊天工具、文档、邮件或其他应用中。

#### 5.4 保存贴图

点击贴图窗口上的“保存”按钮，可将当前截图保存为 PNG 文件。

默认保存目录位于：

- `图片/SLSnap`

如果你已在设置中修改保存目录，则会保存到你指定的位置。

#### 5.5 关闭贴图

点击贴图窗口上的“关闭”按钮，可关闭当前贴图。

### 6. 托盘菜单功能

右键点击系统托盘中的 SLSnap 图标，可以看到以下常用功能：

- `截图`：开始新一次截图
- `设置`：修改快捷键、保存目录、开机启动等选项
- `语言`：切换中文或英文界面
- `关闭全部贴片`：一次性关闭当前所有贴图
- `保存全部贴片`：一次性将当前所有贴图保存到默认目录
- `关于`：查看产品简介
- `退出`：退出程序

### 7. 设置说明

在“设置”中，你可以调整以下内容：

#### 7.1 截图快捷键

用于触发截图操作的全局快捷键。修改后，无需切换到 SLSnap 窗口，也可以直接开始截图。

#### 7.2 保存目录

用于保存截图文件的默认目录。建议设置为你日常容易找到的位置。

#### 7.3 开机自动启动

开启后，SLSnap 会在 Windows 启动时自动运行，适合高频使用场景。

#### 7.4 语言切换

支持中文和英文两种界面语言。

### 8. 常见使用建议

- 如果你需要边看截图边录入数据，可以把贴图拖到屏幕角落，避免遮挡工作区域
- 如果需要同时对比多张内容，可以连续截图，生成多个贴图并排查看
- 如果只是临时转发截图，优先使用“复制”会更高效
- 如果需要留档或提交问题反馈，优先使用“保存”

### 9. 常见问题

#### 9.1 按快捷键没有反应怎么办？

可能原因：

- 快捷键已被其他软件占用
- 程序未成功启动
- 程序已在托盘运行，但你没有注意到图标

建议处理方式：

- 打开托盘菜单，进入“设置”更换一个新的快捷键
- 确认系统托盘中是否存在 SLSnap 图标
- 退出后重新启动程序

#### 9.2 保存的图片在哪里？

默认情况下，截图会保存到：

- `图片/SLSnap`

如果你修改过保存目录，请以设置中的路径为准。

#### 9.3 为什么会有多个贴图同时存在？

这是 SLSnap 的设计方式。每次确认截图后，都会生成一个新的贴图，方便你同时参考多份内容。

如果你想一次性清理，可以在托盘菜单中使用“关闭全部贴片”。

## English Version

### 1. Product Overview

SLSnap is a Windows desktop screenshot sticker tool.

It lets you capture part of your screen and instantly turn the result into an always-on-top floating sticker window, so you can keep important content visible while working in other applications.

### 2. Typical Use Cases

- Fill in forms while referencing a screenshot
- Compare multiple images, tables, web pages, or design drafts
- Keep a temporary screenshot visible on the desktop
- Copy screenshots directly into chat tools, documents, or emails
- Save screenshots locally for records or feedback

### 3. How to Start

After SLSnap starts, it stays in the Windows system tray.

- Double-click the tray icon: start a capture immediately
- Right-click the tray icon: open the function menu
- Default global capture hotkey: `Ctrl+Alt+A`

> Note: If the default hotkey conflicts with another app, you can change it in Settings.

### 4. Quick Start

#### 4.1 Start a Capture

You can start a capture in any of the following ways:

- Press the global hotkey `Ctrl+Alt+A`
- Double-click the SLSnap tray icon
- Right-click the tray icon and choose `Capture`

After that, the screen enters selection mode.

#### 4.2 Select an Area

- Drag with the left mouse button to create a capture area
- Drag the edges or corners to resize the selection
- Drag inside the selection to move it
- Hold `Shift` while resizing to keep the aspect ratio
- Press `Enter` to confirm
- Press `Esc` to cancel
- Double-click inside the selected area to confirm directly

#### 4.3 Create a Desktop Sticker

Once confirmed, SLSnap creates a desktop sticker window immediately. The sticker stays on top so you can keep it visible while you work.

### 5. How to Use a Sticker Window

Each sticker can be used independently.

#### 5.1 Move the Sticker

Drag the top area of the sticker window to move it.

#### 5.2 Zoom the Sticker

Move the mouse over the image area and use the mouse wheel to zoom in or out.

#### 5.3 Copy the Sticker

Click the `Copy` button to copy the current screenshot to the clipboard, so you can paste it into chat tools, documents, emails, or other apps.

#### 5.4 Save the Sticker

Click the `Save` button to save the current screenshot as a PNG file.

The default save folder is:

- `Pictures/SLSnap`

If you changed the save folder in Settings, the image will be saved to your custom location.

#### 5.5 Close the Sticker

Click the `Close` button to close the current sticker.

### 6. Tray Menu Functions

Right-click the SLSnap tray icon to access these functions:

- `Capture`: start a new capture
- `Settings`: change the hotkey, save folder, startup option, and more
- `Language`: switch between Chinese and English
- `Close All Stickers`: close all current stickers at once
- `Save All Stickers`: save all current stickers to the default folder
- `About`: view product information
- `Exit`: quit the application

### 7. Settings

In `Settings`, you can adjust the following items:

#### 7.1 Capture Hotkey

This is the global shortcut used to start a capture. After you change it, you can trigger capture without switching to the SLSnap window.

#### 7.2 Save Directory

This is the default folder for saved screenshots. It is recommended to choose a location that is easy to access in daily work.

#### 7.3 Launch at Windows Startup

When enabled, SLSnap starts automatically with Windows. This is useful if you use it frequently.

#### 7.4 Language

The interface supports both Chinese and English.

### 8. Practical Tips

- If you need to enter data while looking at a screenshot, place the sticker near a corner of the screen to reduce interference
- If you need to compare multiple pieces of content, create several stickers and arrange them side by side
- If you only need to send the screenshot quickly, `Copy` is usually faster than saving
- If you need a file for archive or feedback, use `Save`

### 9. FAQ

#### 9.1 What if the hotkey does not work?

Possible reasons:

- The hotkey is already used by another app
- The application did not start correctly
- The application is already running in the tray, but the icon was overlooked

Suggested actions:

- Open the tray menu and change the hotkey in `Settings`
- Check whether the SLSnap icon is visible in the system tray
- Exit the app and start it again

#### 9.2 Where are saved images stored?

By default, images are saved in:

- `Pictures/SLSnap`

If you changed the save directory, use the path configured in `Settings`.

#### 9.3 Why do multiple stickers stay on screen at the same time?

This is the intended behavior of SLSnap. Every confirmed capture creates a new sticker, making it easier to compare multiple pieces of content at once.

If you want to clear them all quickly, use `Close All Stickers` in the tray menu.
