# SLSnap 隐私政策

生效日期：2026 年 5 月 2 日

SLSnap 是一款本地运行的 Windows 截图贴图工具。我们重视用户隐私，并尽量减少数据收集。本隐私政策说明 SLSnap 如何处理与截图、剪贴板和本地保存相关的信息。

## 一、我们处理的信息

SLSnap 会在用户主动发起截图时读取当前屏幕画面，并截取用户选择的区域。截取的图片仅用于以下本地功能：

- 生成桌面贴片窗口。
- 将截图写入 Windows 系统剪贴板。
- 在用户主动保存时，将截图保存到本地目录。

SLSnap 不会在后台自动截屏。只有在用户通过快捷键、托盘菜单或其他明确操作发起截图时，应用才会进入截图流程。

## 二、我们不收集的信息

当前版本的 SLSnap 不收集、上传、出售或共享以下信息：

- 截图内容。
- 剪贴板内容。
- 本地文件内容。
- 账号信息。
- 联系人信息。
- 浏览历史。
- 精确地理位置。
- 用户身份标识。
- 使用行为分析数据。

## 三、截图内容如何保存

截图内容默认只存在于用户本机，包括：

- 应用内存中的贴片图片。
- Windows 系统剪贴板中的图片。
- 用户主动保存到本地目录的 PNG 文件。

如果用户没有主动保存截图，SLSnap 不会把截图写入文件。关闭贴片或退出应用后，贴片窗口中的图片会随应用状态释放。

## 四、剪贴板使用

当用户确认截图并生成贴片时，SLSnap 会自动把截取的图片写入 Windows 系统剪贴板，方便用户粘贴到聊天工具、文档、邮件或其他应用中。

用户也可以通过贴片窗口中的复制按钮再次将当前贴片图片写入剪贴板。

请注意，剪贴板属于 Windows 系统功能，其他应用可能在用户授权或系统允许的情况下读取剪贴板内容。用户可以通过复制其他内容或清空剪贴板来覆盖 SLSnap 写入的图片。

## 五、本地保存

当用户点击保存或使用保存全部贴片功能时，SLSnap 会把图片保存到用户设置或选择的本地目录。保存后的文件由用户自行管理，用户可以随时删除、移动或备份这些文件。

SLSnap 不会主动上传这些文件到云端服务器。

## 六、网络传输

当前版本的 SLSnap 不需要账号登录，不依赖云端服务，也不会主动将截图、剪贴板内容或本地文件发送到开发者服务器。

如果未来版本增加联网、账号、云同步、崩溃上报或数据分析功能，我们会在功能上线前更新本隐私政策，并在必要时征得用户同意。

## 七、第三方服务

当前版本的 SLSnap 不集成第三方广告、统计分析或云存储服务。

应用运行依赖 Windows 桌面环境和 .NET Framework 相关组件，这些组件由 Microsoft 提供，并受 Microsoft 相关条款和隐私政策约束。

## 八、用户控制

用户可以通过以下方式控制自己的数据：

- 取消截图操作。
- 关闭单个贴片。
- 关闭全部贴片。
- 退出 SLSnap。
- 删除已保存的本地截图文件。
- 复制其他内容以覆盖系统剪贴板。
- 使用 Windows 系统设置管理剪贴板历史记录。

## 九、儿童隐私

SLSnap 面向一般 Windows 桌面用户，不专门面向儿童。当前版本不收集用户个人信息，也不提供面向儿童的在线服务。

## 十、政策更新

我们可能会根据产品功能变化、法律法规要求或商店政策要求更新本隐私政策。更新后的隐私政策会随应用或公开页面同步发布。
