# SLSnap

Native Windows implementation of the screenshot sticker utility.

## Build

```powershell
.\build-winscreen.ps1
```

Output:

- `dist\winscreen\SLSnap.exe`

## Notes

- Uses WinForms on .NET Framework 4.8
- No Python runtime dependency
- Smaller distribution than the PySide build
